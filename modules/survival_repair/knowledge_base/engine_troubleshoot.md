# Engine Won't Start Guide
## Dead Battery
1. Check battery terminals for corrosion - clean with baking soda + water
2. Try jump start - connect red to positive, black to negative
3. Let run 20+ minutes to recharge
4. If battery dies again - alternator may be faulty

## No Fuel
1. Check fuel gauge - reserve light means 10-15% left
2. Check fuel pump - listen for hum when key is on
3. Prime diesel engine - turn key on/off 3 times before starting
4. Check fuel filter - clogged filter starves engine

## Overheating Shutdown
1. Let cool completely before restarting - 30+ minutes
2. Check coolant level - never open hot radiator cap
3. Check for coolant leaks under vehicle
4. Check radiator fan is working

## Starter Motor Clicks
1. Single click = dead battery
2. Rapid clicks = low battery, try jump start
3. No sound = check fuse, check starter relay
4. Grinding = starter gear worn, avoid repeated attempts
