# Wayward AI
Offline travel assistant for vanlife, RV, and overlanding.

## ⚠️ Important Disclaimer
This tool is for emergency reference only. Information may be incomplete or incorrect.
- Always verify critical information with local sources
- Seek professional help when possible
- Author is not responsible for decisions made based on this data
- Do not rely solely on this tool for safety-critical decisions

## Features
- Offline risk analysis for 6 regions
- DIY repair and survival guides
- Green routing with CO2 calculator
- Community spot sync
- Works offline via Ollama, online via Claude API

## Supported Languages
English, Spanish, Russian

## Regions
Caucasus, Central Asia, Latin America, Africa, Oceania, Siberia
